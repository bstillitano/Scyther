# ``Scyther``

A comprehensive iOS debugging toolkit that helps you cut through bugs in your iOS app.

## Overview

Scyther provides a suite of debugging tools for iOS developers, QA testers, and designers. Simply shake your device to access a powerful debug menu with features for network inspection, feature flags, location spoofing, and more.
