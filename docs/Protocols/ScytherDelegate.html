ScytherDelegate

public protocol ScytherDelegate : AnyObject

Optional delegate that can be implemented to allow an app/caller to know that certain actions have been performed. This protocolis not required to be implemented in order for Scyther to work properly, it should be used primarily as a UI convenience; i.e. When switching environments, show a UIAlertController
